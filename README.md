# Tensorflow-
记录一些知识和学习记录


对于任何一个深度学习库，如mxnet、tensorflow、theano、caffe等，基本上我都采用同样的一个学习流程，大体流程如下：

<p>(1)训练阶段：数据打包-》网络构建、训练-》模型保存-》可视化查看损失函数、验证精度</p>

<p>(2)测试阶段：模型加载-》测试图片读取-》预测显示结果</p>

<p>(3)移植阶段：量化、压缩加速-》微调-》C++移植打包-》上线</p>
